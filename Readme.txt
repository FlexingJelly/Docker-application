Работу сдавал студент Чукаев из группы 20ПМИ-1, рассчитывая на оценку 6-7 баллов 
Запускается без параметров, при запуске загружает и впоследствии использует python-alpine,
копирует все файлы из директории (в нашем случае файл один - hello.py) в папку образа 
и в python-е выполняет программу hello.py.
Программа просит пользователя ввести имя, после чего приветствует его
